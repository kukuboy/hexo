快速部署

搭建环境

这里步骤太简单，而且网上的步骤很多，这里不再介绍

  • node.js环境
  • git 环境

安装hexo

powerShell环境

//全局安装
npm install hexo-cli -g

创建自己的博客文件

powerShell环境

hexo init blogDemo

hexo3.0服务器进行了独立，如果需要这里可以进行安装
cd .\blogDemo\
npm install hexo-server --save


    .
    ├── _config.yml  //主要配置文件
    ├── package.json // 插件管理文件
    ├── scaffolds // 文件模板管理
    ├── source
    |   ├── _drafts
    |   └── _posts // 新建的文章在这里
    └── themes // 主体相关文件

本地运行

powerShell环境

hexo s // 或者hexo server
// 之后便可以在http://localhost:4000/打开
// hexo -s -p 3000 更改端口

新建文章

powerShell环境

hexo new 这是新文章的名字

提交与部署

  • coding
    1. 创建hexoDemo项目，创建代码仓库
    2. 安装hexo-deployer-git,用于git方式的部署
    3. 复制仓库地址，如git@e.coding.net:kukuboy/hexodemo/hexoDemo.git
    4. 将文件主目录下的_confg.yml中的deploy，修改为如下
    5. 执行部署
    6. coding打开持续部署功能，如果没有可在项目设置功能中打开
    7. 选择静态网站托管，选择主目录部署
    8. 部署完成后即可看到指定可访问的域名
